lightsail container serviceで保存できるイメージ数の上限

公開日: 2022/10/08 22:37:51
更新日: 2022/10/08 23:04:38

結論

最大150個

確認

ECRを使わない場合にここに溜まっていく

% aws lightsail push-container-image --service-name playground --label app --image playground-app
