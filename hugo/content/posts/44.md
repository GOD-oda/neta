一番行数が多いファイルを抜き出す
```
% find app -name "*.php" | xargs wc -l | sort -n | tail -n 2
      66 app/Http/Kernel.php
     544 total
```
降順にしてheadコマンドでも良い
```
% find app -name "*.php" | xargs wc -l | sort -r | head -n 2
     544 total
      66 app/Http/Kernel.php
```
