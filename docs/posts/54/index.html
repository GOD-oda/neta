slow_query_log確認コマンド

公開日: 2021/07/20 17:21:05
更新日: 2022/09/12 14:58:51

TABLEでのログ

前提

  1. slow_query_logがONになていること
mysql> show variables like '%slow_query%';
+---------------------+----------------------------------------------+
| Variable_name       | Value                                        |
+---------------------+----------------------------------------------+
| slow_query_log      | ON                                           |
| slow_query_log_file | /rdsdbdata/log/slowquery/mysql-slowquery.log |
+---------------------+----------------------------------------------+
2 rows in set (0.00 sec)
  1. log_outputがTABLEになっていること
mysql> show variables like '%output%';
+---------------+-------+
| Variable_name | Value |
+---------------+-------+
| log_output    | TABLE |
+---------------+-------+
1 row in set (0.00 sec)

コマンド

上位10件を抽出する例

mysql> select * from mysql.slow_log order by query_time desc limit 10\G

    start_time: 2021-07-16 14:00:28

備考

スロークエリの定義 下記だと2秒以上かかったクエリをスロークエリとみなす

mysql> show variables like '%query_time%';
+-----------------+----------+
| Variable_name   | Value    |
+-----------------+----------+
| long_query_time | 2.000000 |
+-----------------+----------+
1 row in set (0.00 sec)
# mysql5.6
